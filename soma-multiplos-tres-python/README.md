# 📌 Sistema de Soma de Múltiplos de 3

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema recebe números informados pelo usuário, filtra apenas os valores divisíveis por 3 e realiza automaticamente a soma desses números.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, operadores matemáticos e controle de fluxo, desenvolvendo uma solução capaz de analisar entradas, selecionar valores específicos e realizar operações apenas com dados que atendem a determinados critérios.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Verifica se cada número é divisível por 3;
- Ignora valores que não atendem ao critério;
- Soma apenas os números múltiplos de 3;
- Encerra a execução quando o valor informado for **0**;
- Exibe a soma final dos valores válidos.

Regras aplicadas:

- Número divisível por 3:
  - É adicionado ao cálculo da soma.

- Número não divisível por 3:
  - É ignorado pelo sistema.

- Número **0**:
  - Finaliza a entrada de dados.

## ✨ Funcionalidades

- Entrada contínua de números;
- Filtragem automática de valores;
- Soma de valores específicos;
- Controle de encerramento;
- Processamento de dados conforme regras definidas.
