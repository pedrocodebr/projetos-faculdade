# atividade2-python

## Descrição
Este projeto tem como objetivo contar a quantidade de números pares em uma lista específica.

## Objetivo
A intenção deste projeto é praticar o uso de estruturas de repetição e condicionais em Python para realizar uma tarefa simples, aumentando assim o entendimento sobre lógica de programação.

## O que foi praticado/aprendido
- Estrutura de loop `for`
- Condicional if/else
- Operador módulo `%` para verificar números pares

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Baixe o arquivo `atividade2.py`.
2. Abra um terminal ou prompt de comando.
3. Navegue até o diretório onde o arquivo foi salvo.
4. Execute o seguinte comando:

```sh
python atividade2.py
```

Este projeto não requer nenhum pacote externo.
