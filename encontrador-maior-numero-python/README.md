# at13-python

## Descrição
Este projeto é uma simples aplicação em Python que lê números inteiros do usuário e determina o maior número entre eles. O programa continua solicitando números até que o usuário digite zero, após isso exibe o maior número informado.

## Objetivo
O objetivo deste projeto era praticar a leitura de entrada do usuário, condicionais, laços de repetição (while) e operações básicas em Python.

## O que foi praticado/aprendido
- Leitura de entrada do usuário usando `input()`
- Uso da estrutura de controle `if` para comparação
- Loop infinito (`while True`) com condição de saída (`break`)
- Variáveis e seus tipos (inicialização, reatribuição)
- Operadores comparativos

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Salve o código em um arquivo chamado `at13.py`.
2. Abra um terminal ou prompt de comando.
3. Navegue até o diretório onde salvou o arquivo.
4. Execute o script digitando:
   ```bash
   python at13.py
   ```
5. Siga as instruções na tela para digitar os números e encontrar o maior.
