# at10-python

## Descrição
Este projeto é uma aplicação simples que simula um sistema de votação entre dois candidatos e conta os votos válidos e nulos.

## Objetivo
O objetivo do projeto foi praticar a criação de loops, condicionais e variáveis em Python para resolver um problema prático de contagem de votos.

## O que foi praticado/aprendido
- Uso do loop `while` para processar entrada do usuário até uma condição específica ser atendida.
- Condicionais (`if`, `elif`, `else`) para decidir qual ação tomar com base nos dados de entrada.
- Incremento e atribuição em variáveis para contar os votos.
- Estrutura condicional para determinar o vencedor ou se houve empate.

## Tecnologias utilizadas
- Python

## Como executar
1. Salve o código acima em um arquivo chamado `at10.py`.
2. Abra um terminal e navegue até o diretório onde o arquivo foi salvo.
3. Execute o script digitando `python at10.py` no terminal.

Após a execução, o programa irá solicitar que você vote em um dos candidatos (1 ou 2), ou vote nulo (3). Quando desejar encerrar a votação, digite 0. Após isso, o programa exibirá o resultado da votação.
