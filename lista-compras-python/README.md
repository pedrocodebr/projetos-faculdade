# atividade3-python

## Descrição
Este projeto é uma introdução básica ao uso de loops e listas em Python. Ele imprime a lista de compras e exibe a quantidade total de produtos nessa lista.

## Objetivo
O objetivo deste projeto era demonstrar como criar e manipular listas em Python, bem como usar um loop `for` para iterar sobre os elementos da lista.

## O que foi praticado/aprendido
- Manipulação de listas em Python
- Uso do loop `for`
- Impressão de saída na tela
- Função `len()` para obter a quantidade de elementos em uma lista

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Certifique-se de ter o Python instalado no seu sistema.
2. Salve o código acima em um arquivo chamado `atividade3.py`.
3. Abra um terminal ou prompt de comando e navegue até o diretório onde salvou o arquivo.
4. Execute o seguinte comando:
   ```
   python atividade3.py
   ```

Após a execução, você verá a lista de compras sendo impressa na tela, seguida pela quantidade total de produtos na lista.
