# 📌 Sistema de Contagem de Números Pares e Ímpares

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite registrar diversos números inteiros informados pelo usuário e contabiliza automaticamente quantos deles são pares e quantos são ímpares.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, estruturas condicionais e operadores matemáticos, por meio da implementação de um sistema capaz de classificar números de acordo com sua paridade.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Permite o cadastro contínuo de valores;
- Verifica se cada número é par ou ímpar;
- Atualiza automaticamente os contadores;
- Encerra a execução quando um número negativo é informado;
- Exibe a quantidade total de números pares e ímpares registrados.

### Regras aplicadas

- **Número par:**
  - Incrementa o contador de números pares.

- **Número ímpar:**
  - Incrementa o contador de números ímpares.

- **Número negativo:**
  - Finaliza a execução do programa.

## ✨ Funcionalidades

- Cadastro contínuo de números;
- Classificação automática em pares e ímpares;
- Contagem de ocorrências;
- Encerramento controlado do sistema;
- Exibição de resumo estatístico.
