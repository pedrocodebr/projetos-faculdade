# 📌 Sistema de Análise de Maior e Menor Valor

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite registrar diversos números informados pelo usuário e identificar automaticamente o maior e o menor valor dentre os dados válidos.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, estruturas condicionais, validação de dados e comparação de valores, por meio da implementação de um sistema simples de análise numérica.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Permite o cadastro contínuo de valores;
- Ignora números negativos;
- Encerra a execução quando o usuário informa **0**;
- Identifica automaticamente o maior valor informado;
- Identifica automaticamente o menor valor informado;
- Exibe os resultados ao final da execução.

### Regras aplicadas

- **Número positivo:**
  - É considerado válido para análise.

- **Número negativo:**
  - É ignorado pelo sistema.

- **Valor 0:**
  - Finaliza a entrada de dados.

## ✨ Funcionalidades

- Cadastro contínuo de números;
- Validação automática dos dados;
- Identificação do maior valor;
- Identificação do menor valor;
- Processamento em tempo real dos dados;
- Exibição de um resumo ao final da execução.
