# at12-python

## Descrição
Este projeto imprime os números ímpares menores que 50, com a exclusão de qualquer número divisível por 7.

## Objetivo
O objetivo deste projeto é praticar a manipulação de loops e condições em Python, além de aprender sobre o uso do `continue` e `break`.

## O que foi praticado/aprendido
- Estruturas de loop (`for`)
- Condições (`if`, `elif`, `else`)
- Operadores aritméticos (`%`)
- Comandos de controle de fluxo (`continue`, `break`)

## Tecnologias utilizadas
- Python 3

## Como executar
1. Certifique-se de ter o Python instalado em sua máquina.
2. Baixe o arquivo `at12.py`.
3. Abra um terminal ou prompt de comando.
4. Navegue até o diretório onde você salvou o arquivo.
5. Execute o script digitando:
   ```
   python at12.py
   ```

Este projeto foi criado para auxiliar na compreensão e aplicação prática dos fundamentos do Python em manipulação de fluxo de dados.
