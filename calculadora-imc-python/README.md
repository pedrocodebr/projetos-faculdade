# at14-python

## Descrição
Este projeto calcula e categoriza a quantidade de pessoas cujo IMC (Índice de Massa Corporal) está abaixo, normal ou acima dos padrões recomendados.

## Objetivo
O objetivo deste projeto é praticar a programação em Python, desenvolver uma lógica condicional e utilizar estruturas de repetição para processamento de dados inseridos pelo usuário.

## O que foi praticado/aprendido
- Conceitos de variáveis e operações matemáticas.
- Estrutura de loop `while` e controle de fluxo (`break`, `continue`).
- Criação de condições lógicas com `if`, `elif`, e `else`.
- Formatação e exibição de resultados no console.

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Certifique-se de ter o Python instalado em seu sistema.
2. Salve o código fornecido em um arquivo chamado `at14.py`.
3. Abra um terminal ou prompt de comando.
4. Navegue até o diretório onde o arquivo `at14.py` está localizado.
5. Execute o script com o seguinte comando:
   ```
   python at14.py
   ```
6. Siga as instruções na tela para inserir os pesos e alturas dos indivíduos.
