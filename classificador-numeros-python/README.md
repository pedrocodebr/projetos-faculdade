# 📌 Sistema de Classificação de Números

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza a análise de uma sequência de números informados pelo usuário, classificando-os conforme suas características, como paridade e sinal.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, variáveis acumuladoras e operadores matemáticos, desenvolvendo uma solução capaz de processar diversos valores e apresentar uma análise dos resultados obtidos.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita 10 números ao usuário;
- Verifica se cada número é:
  - **Par ou ímpar**;
  - **Positivo ou negativo**;
- Armazena a quantidade de cada classificação;
- Exibe um resumo final com os resultados.

Classificações realizadas:

- Números pares;
- Números ímpares;
- Números positivos;
- Números negativos.

## ✨ Funcionalidades

- Entrada de múltiplos números;
- Classificação automática dos valores informados;
- Contagem de números pares e ímpares;
- Contagem de números positivos e negativos;
- Exibição de um resumo final da análise.
