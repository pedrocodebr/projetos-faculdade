# 📌 Filtro de Números Ímpares com Controle de Loop

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O programa percorre uma sequência de números, aplica filtros utilizando condições específicas e controla a execução do laço de repetição conforme determinadas regras.

## 🎯 Objetivo

O objetivo deste projeto foi praticar estruturas de repetição e comandos de controle de fluxo, desenvolvendo uma solução capaz de analisar valores sequenciais e executar ações específicas durante a iteração.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Percorre números de 1 até 50;
- Ignora números pares;
- Analisa apenas números ímpares;
- Encerra a execução quando encontra um número ímpar divisível por 7;
- Exibe os números processados antes da interrupção.

Regras aplicadas:

- Número par:
  - É ignorado utilizando o comando `continue`.

- Número ímpar divisível por 7:
  - Encerra o processamento utilizando o comando `break`.

- Outros números ímpares:
  - São exibidos no terminal.

## ✨ Funcionalidades

- Percorrimento automático de sequência numérica;
- Filtro de números pares;
- Identificação de múltiplos de 7;
- Controle de interrupção do laço;
- Processamento condicional de valores.
