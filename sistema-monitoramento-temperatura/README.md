# 📌 Sistema de Classificação de Temperatura

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza a classificação de uma temperatura informada pelo usuário, identificando diferentes níveis climáticos e exibindo alertas de acordo com a faixa de temperatura registrada.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como entrada e saída de dados, comparação de valores e estruturas condicionais, criando um sistema capaz de analisar informações e retornar uma classificação automaticamente.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita a temperatura atual da cidade;
- Exibe a temperatura informada pelo usuário;
- Analisa a temperatura de acordo com faixas definidas;
- Classifica a condição climática encontrada.

Classificações utilizadas:

- **Temperatura acima de 40°C:** Emergência climática;
- **Temperatura acima de 35°C:** Calor extremo;
- **Temperatura entre 26°C e 35°C:** Calor moderado;
- **Temperatura entre 10°C e 25°C:** Temperatura amena;
- **Temperatura abaixo de 10°C:** Frio intenso.

## ✨ Funcionalidades

- Leitura da temperatura informada pelo usuário;
- Classificação automática das condições climáticas;
- Exibição de alertas conforme a temperatura;
- Formatação da temperatura com uma casa decimal.

## 🛠️ Tecnologias Utilizadas

- Python 3

## 🧠 Conceitos Praticados

- Variáveis;
- Entrada e saída de dados;
- Conversão de tipos (`float`);
