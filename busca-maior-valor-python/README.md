# 📌 Sistema de Busca do Maior Valor

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite a entrada de diversos números e identifica automaticamente qual foi o maior valor informado pelo usuário.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, comparação de valores e inicialização de variáveis, desenvolvendo uma solução capaz de analisar uma sequência de dados e retornar o maior elemento encontrado.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Armazena temporariamente o maior valor encontrado;
- Compara cada novo número informado com o maior valor atual;
- Atualiza o resultado quando encontra um valor superior;
- Encerra a execução quando o usuário informa **0**;
- Exibe o maior número encontrado.

Regras aplicadas:

- Número informado:
  - É comparado com o maior valor atual.

- Valor maior que o armazenado:
  - Atualiza o maior valor encontrado.

- Número **0**:
  - Finaliza a entrada de dados.

## ✨ Funcionalidades

- Entrada contínua de números;
- Comparação automática de valores;
- Identificação do maior número informado;
- Controle de encerramento;
- Processamento de dados em tempo real.
