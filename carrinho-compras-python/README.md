# at8-python

## Descrição
Este é um programa simples em Python que calcula o total de compras e a quantidade de descontos aplicados. O usuário pode inserir os valores das suas compras e, se o valor for maior que R$100,00, será aplicado um desconto de 10%. O programa continua solicitando novos valores até que o usuário insira -1.

## Objetivo
O objetivo deste projeto é praticar a lógica de programação em Python, incluindo a utilização de loops `while`, condicionais, e entrada/saída de dados.

## O que foi praticado/aprendido
- Conhecimento sobre laços de repetição (`while`)
- Condicionais (`if`, `elif`, `else`)
- Entrada de dados pelo usuário (`input()`)
- Operações aritméticas em Python
- Estrutura de controle para saída do loop

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Certifique-se de ter o Python instalado na sua máquina.
2. Crie um novo arquivo chamado `at8.py`.
3. Copie e cole o código fornecido acima em `at8.py`.
4. Abra o terminal ou prompt de comando.
5. Navegue até o diretório onde está localizado `at8.py`.
6. Execute o programa digitando `python at8.py` no terminal.

O programa iniciará solicitando que você insira o valor da compra. Digite `-1` para encerrar a entrada de dados e obter os resultados finais.
