# at9-python

## Descrição
Este projeto verifica números inteiros para determinar se são primos e conta a quantidade total de números primos digitados pelo usuário.

## Objetivo
O objetivo deste projeto é praticar a programação em Python, desenvolver uma função que verifique se um número é primo e implementar um loop para coletar múltiplos números até que o usuário decida sair.

## O que foi praticado/aprendido
- Lógica de programação condicional.
- Funções em Python.
- Estrutura de loop `while`.
- Operações matemáticas básicas.
- Contagem de valores.

## Tecnologias utilizadas
- Python

## Como executar
1. Certifique-se de ter o Python instalado na sua máquina.
2. Salve o código acima em um arquivo chamado `at9.py`.
3. Abra uma linha de comando e navegue até o diretório onde salvou o arquivo.
4. Execute o script digitando `python at9.py` e pressione Enter.

O programa solicitará que você insira números inteiros. Ele continuará pedindo para inserir números até que você digite `1`. Após isso, ele exibirá a quantidade total de números primos que foram inseridos.
