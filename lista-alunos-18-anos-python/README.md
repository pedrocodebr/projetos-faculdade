# atividade4-python

## Descrição
Este projeto tem como objetivo imprimir os nomes dos alunos que têm 18 anos ou mais.

## Objetivo
A intenção deste projeto é demonstrar a lógica condicional em Python e manipulação de listas de tuplas.

## O que foi praticado/aprendido
- Lógica condicional (`if`)
- Estrutura de repetição (`for`)
- Manipulação de listas e tuplas

## Tecnologias utilizadas
- Python

## Como executar
1. Salve o código acima em um arquivo chamado `atividade4.py`.
2. Abra uma linha de comando.
3. Navegue até o diretório onde o arquivo foi salvo.
4. Execute o script com o seguinte comando: `python atividade4.py`.

O projeto deve exibir os nomes "Ana" e "Carlos", conforme as idades informadas na lista.
