# atividade9-python

## Descrição
Este projeto contém um script simples em Python que percorre uma lista de vendas e imprime apenas as vendas maiores que 100.

## Objetivo
O objetivo deste projeto é praticar a estrutura de repetição `for` e o uso de condicionais em Python, além de mostrar como acessar elementos de uma lista.

## O que foi praticado/aprendido
- Estrutura de repetição `for`
- Condicionais (`if`)
- Acesso a elementos de uma lista

## Tecnologias utilizadas
- Linguagem de programação: Python

## Como executar
1. Salve o código acima em um arquivo chamado `atividade9.py`.
2. Abra um terminal ou prompt de comando.
3. Navegue até o diretório onde o arquivo foi salvo.
4. Execute o script digitando o seguinte comando:
   ```bash
   python atividade9.py
   ```
5. O programa imprimirá as vendas maiores que 100 na lista `vendas`.
