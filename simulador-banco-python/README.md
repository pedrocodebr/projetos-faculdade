# 📌 Sistema de Caixa Eletrônico em Python

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema simula um caixa eletrônico, permitindo que o usuário realize operações bancárias básicas como consulta de saldo, saque e depósito através de um menu interativo.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de lógica de programação, estruturas de repetição, estruturas condicionais e manipulação de valores, desenvolvendo uma aplicação capaz de controlar operações financeiras simples e manter informações atualizadas durante sua execução.

## 📝 Sobre o Projeto

O programa inicia com um saldo padrão e disponibiliza um menu de operações:

- Consultar saldo;
- Realizar saque;
- Realizar depósito;
- Encerrar o sistema.

Operações disponíveis:

### 💰 Consulta de saldo
- Exibe o saldo atual da conta formatado em reais.

### 💸 Saque
- Solicita o valor desejado;
- Verifica se o valor é válido;
- Confere se existe saldo suficiente;
- Atualiza o saldo após a operação.

### 📥 Depósito
- Solicita o valor a ser depositado;
- Valida se o valor é permitido;
- Atualiza o saldo da conta.

### 🚪 Encerramento
- Permite sair do sistema através da opção de saída.

## ✨ Funcionalidades
