# atividade5-python

## Descrição
Este projeto apresenta um simples programa em Python que cria uma lista de produtos e seus preços.

## Objetivo
O objetivo deste projeto era praticar a manipulação de dicionários em Python, aprendendo como iterar sobre eles para exibir informações específicas.

## O que foi praticado/aprendido
- Manipulação de estruturas de dados em Python, especificamente dicionários.
- Uso do laço `for` para iteração sobre itens de um dicionário.
- Formatação de strings para exibição de valores numéricos.

## Tecnologias utilizadas
- Linguagem de programação: Python

## Como executar
1. Salve o código em um arquivo chamado `atividade5.py`.
2. Abra um terminal ou prompt de comando.
3. Navegue até o diretório onde o arquivo foi salvo.
4. Execute o programa com o seguinte comando:
   ```
   python atividade5.py
   ```

Este projeto foi criado como parte de uma atividade prática, fornecendo uma introdução básica ao trabalho com dicionários em Python.
