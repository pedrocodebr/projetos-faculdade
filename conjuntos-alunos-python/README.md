# atividade10-python

## Descrição
Este projeto demonstra como usar conjuntos (sets) em Python para encontrar elementos comuns e exclusivos entre duas listas.

## Objetivo
O objetivo deste projeto é praticar a manipulação de conjuntos em Python, incluindo operações como interseção (`&`) e diferença (`-`), para analisar as relações entre dois grupos (turmas).

## O que foi praticado/aprendido
- Operações com conjuntos em Python: `&`, `-`
- Conversão de listas em conjuntos usando `set()`
- Utilização do método `print()` para exibir resultados

## Tecnologias utilizadas
- Linguagem de programação: Python

## Como executar
1. Salve o código acima em um arquivo chamado `atividade10.py`.
2. Abra um terminal ou prompt de comando.
3. Navegue até o diretório onde o arquivo foi salvo.
4. Execute o projeto com o seguinte comando:
   ```
   python atividade10.py
   ```

Este projeto é uma introdução prática ao uso de conjuntos em Python para análise de dados simples.
