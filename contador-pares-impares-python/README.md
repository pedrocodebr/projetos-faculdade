# 📌 Contador de Números Pares e Ímpares

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza a leitura de números informados pelo usuário e classifica cada valor como par ou ímpar, armazenando a quantidade de ocorrências de cada categoria.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, operadores matemáticos e variáveis contadoras, desenvolvendo uma solução capaz de analisar múltiplos valores e gerar um resumo dos resultados obtidos.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números ao usuário continuamente;
- Verifica se cada número informado é par ou ímpar;
- Realiza a contagem de cada classificação;
- Encerra a execução quando um número negativo é informado;
- Exibe a quantidade total de números pares e ímpares encontrados.

Regras aplicadas:

- Número positivo ou zero:
  - É analisado pelo sistema.

- Número par:
  - Incrementa o contador de pares.

- Número ímpar:
  - Incrementa o contador de ímpares.

- Número negativo:
  - Finaliza o programa.

## ✨ Funcionalidades

- Entrada contínua de números;
- Classificação automática entre par e ímpar;
- Contagem dos valores analisados;
- Encerramento através de condição de parada;
- Exibição do resumo final.
