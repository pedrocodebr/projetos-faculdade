# 📌 Sistema de Contagem de Números Primos

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite que o usuário informe diversos números e realiza a identificação automática de quais valores são números primos, contabilizando a quantidade encontrada.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de funções, estruturas de repetição e validação matemática, desenvolvendo uma solução organizada capaz de analisar números e retornar informações específicas.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Verifica se cada número informado é primo;
- Utiliza uma função específica para realizar a validação;
- Conta a quantidade de números primos encontrados;
- Encerra o programa quando o usuário informa o valor **1**;
- Exibe a quantidade total de números primos identificados.

Funcionamento da verificação:

- Números menores que 2:
  - Não são considerados primos.

- Números maiores ou iguais a 2:
  - São testados através de divisões sucessivas.

- Caso possuam apenas dois divisores:
  - São classificados como números primos.

## ✨ Funcionalidades

- Verificação automática de números primos;
- Entrada contínua de valores;
- Contagem de números primos encontrados;
- Organização do código utilizando função;
