# 📌 Sistema de Análise de Valores Válidos

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite a entrada de diversos números, realiza a filtragem dos valores inválidos e identifica automaticamente o maior e o menor número informado.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, validação de dados e comparação de valores, criando um algoritmo capaz de processar entradas continuamente e armazenar apenas informações relevantes para análise.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Permite encerrar a execução informando o valor **0**;
- Ignora números negativos;
- Analisa somente valores positivos;
- Identifica o maior e o menor valor informado;
- Exibe os resultados ao final da execução.

Regras aplicadas:

- Número positivo:
  - É considerado para análise.

- Número negativo:
  - É ignorado pelo sistema.

- Número **0**:
  - Finaliza a entrada de dados.

## ✨ Funcionalidades

- Entrada contínua de números;
- Filtro de valores negativos;
- Identificação do maior valor;
- Identificação do menor valor;
- Controle de encerramento do programa;
- Processamento de dados durante a execução.
