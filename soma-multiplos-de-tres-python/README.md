# 📌 Sistema de Soma de Múltiplos de 3

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite receber diversos números inteiros informados pelo usuário e calcular automaticamente a soma daqueles que são múltiplos de 3.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, estruturas condicionais, operadores matemáticos e controle de fluxo, por meio da implementação de um sistema capaz de filtrar e processar dados de acordo com critérios definidos.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números inteiros ao usuário;
- Permite o cadastro contínuo de valores;
- Verifica se cada número é múltiplo de 3;
- Soma apenas os números que atendem ao critério;
- Ignora automaticamente os demais valores;
- Finaliza a execução quando o usuário informa **0**;
- Exibe a soma dos números múltiplos de 3.

### Regras aplicadas

- **Número múltiplo de 3:**
  - É adicionado à soma.

- **Número que não é múltiplo de 3:**
  - É ignorado pelo sistema.

- **Valor 0:**
  - Encerra a execução do programa.

## ✨ Funcionalidades

- Cadastro contínuo de números;
- Identificação automática de múltiplos de 3;
- Soma acumulada dos valores válidos;
- Filtragem de dados utilizando controle de fluxo;
- Exibição do resultado final.
