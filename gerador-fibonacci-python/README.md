# 📌 Gerador da Sequência de Fibonacci

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema gera a sequência de Fibonacci até um limite informado pelo usuário, utilizando estruturas de repetição e operações matemáticas para calcular os próximos valores da sequência.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, atualização de variáveis e lógica de algoritmos, desenvolvendo uma solução capaz de gerar automaticamente uma sequência numérica baseada em uma regra matemática.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita um limite máximo para a sequência;
- Inicializa os dois primeiros valores da sequência de Fibonacci;
- Calcula os próximos números utilizando a soma dos dois valores anteriores;
- Continua a geração enquanto o valor atual não ultrapassa o limite informado;
- Exibe a sequência gerada.

A sequência de Fibonacci funciona da seguinte forma:

- Cada número é resultado da soma dos dois números anteriores;
- A sequência inicia com:

```
0, 1, 1, 2, 3, 5, 8, 13...
```

## ✨ Funcionalidades

- Geração automática da sequência de Fibonacci;
- Definição de limite personalizado pelo usuário;
- Processamento utilizando estrutura de repetição;
- Exibição dos valores da sequência em ordem.
