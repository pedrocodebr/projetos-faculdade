# 📌 Sistema de Análise de Compras em Python

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza o cadastro de valores de compras informados pelo usuário e gera uma análise dos dados coletados, calculando a média das compras e identificando quantos valores ficaram acima desse resultado.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de listas, estruturas de repetição e processamento de dados, desenvolvendo uma solução capaz de armazenar informações, realizar cálculos e extrair resultados a partir dos valores registrados.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita valores de compras ao usuário;
- Armazena os valores válidos em uma lista;
- Permite encerrar o cadastro informando um valor negativo;
- Ignora valores iguais a zero;
- Calcula a média das compras registradas;
- Analisa quais compras ficaram acima da média;
- Exibe os resultados da análise.

Regras aplicadas:

- Valores maiores que zero:
  - São armazenados para análise.

- Valor igual a zero:
  - É ignorado pelo sistema.

- Valor negativo:
  - Encerra a entrada de dados.

## ✨ Funcionalidades

- Cadastro de múltiplos valores;
- Armazenamento de dados utilizando listas;
- Cálculo automático da média das compras;
- Identificação de compras acima da média;
- Validação de valores informados;
- Geração de resumo dos resultados.
