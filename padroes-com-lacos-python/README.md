# 📌 Gerador de Padrão Numérico em Python

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema gera um padrão numérico em formato de sequência crescente de acordo com a quantidade de linhas informada pelo usuário.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição aninhadas, desenvolvendo a capacidade de controlar múltiplos ciclos de execução e criar padrões utilizando lógica de programação.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita ao usuário a quantidade de linhas desejadas;
- Utiliza um laço de repetição externo para controlar as linhas;
- Utiliza um segundo laço de repetição para controlar os números exibidos em cada linha;
- Exibe um padrão numérico crescente.

Exemplo de execução:

Entrada:
```
5
```

Saída:
```
1
12
123
1234
12345
```

## ✨ Funcionalidades

- Geração automática de padrões numéricos;
- Quantidade de linhas definida pelo usuário;
- Utilização de múltiplos laços de repetição;
- Controle de impressão em diferentes formatos.

## 🛠️ Tecnologias Utilizadas

- Python 3

## 🧠 Conceitos Praticados

- Variáveis;
- Entrada e saída de dados;
- Conversão de tipos (`int`);
- Estruturas de repetição:
  - `for`
  - `range()`
- Laços de repetição aninhados;
- Controle de impressão:
  - `end=""`
- Organização de fluxo de execução;
- Lógica para criação de padrões.
