# 📌 Sistema de Votação e Apuração de Resultados

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema simula uma votação simples, permitindo registrar votos para candidatos, contabilizar votos nulos e apresentar o resultado final da eleição.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, estruturas condicionais e variáveis contadoras, desenvolvendo uma solução capaz de coletar informações e realizar uma apuração automática dos resultados.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita votos continuamente ao usuário;
- Registra votos para candidatos;
- Contabiliza votos nulos;
- Permite encerrar a votação através do valor **0**;
- Compara a quantidade de votos dos candidatos;
- Informa o vencedor ou caso ocorra empate.

Regras aplicadas:

- Voto **1**:
  - Registrado para o candidato 1.

- Voto **2**:
  - Registrado para o candidato 2.

- Voto **3**:
  - Contabilizado como voto nulo.

- Voto **0**:
  - Encerra a votação e inicia a apuração.

## ✨ Funcionalidades

- Registro de votos;
- Contagem automática de candidatos;
- Contagem de votos nulos;
- Apuração do resultado;
- Identificação de empate;
- Sistema de votação contínuo.
