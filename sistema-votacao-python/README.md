# 📌 Sistema de Votação e Apuração de Resultados

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema simula uma votação simples, permitindo o registro de votos para diferentes candidatos e realizando automaticamente a apuração do resultado final.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, variáveis contadoras, estruturas condicionais e lógica de comparação, desenvolvendo uma solução capaz de registrar votos e identificar o candidato vencedor.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Apresenta opções de candidatos para votação;
- Recebe votos informados pelo usuário;
- Contabiliza a quantidade de votos de cada candidato;
- Permite encerrar a votação através de uma opção de saída;
- Exibe o total de votos de cada candidato;
- Verifica qual candidato possui a maior quantidade de votos;
- Informa o vencedor ou caso ocorra empate.

Funcionamento:

- Opção **1**:
  - Registra voto para o candidato 1.

- Opção **2**:
  - Registra voto para o candidato 2.

- Opção **3**:
  - Registra voto para o candidato 3.

- Opção **0**:
  - Encerra a votação e inicia a apuração.

## ✨ Funcionalidades

- Registro de múltiplos votos;
- Contagem individual por candidato;
- Validação de opções de voto;
- Encerramento manual da votação;
- Apuração automática do resultado;
- Identificação de vencedor ou empate.
