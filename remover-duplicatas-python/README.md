# atividade7-python

## Descrição
Este projeto demonstra como remover duplicados de uma lista usando a estrutura de dados `set` em Python.

## Objetivo
O objetivo é aprender a manipular coleções em Python e entender como a conversão de uma lista para um conjunto remove automaticamente os elementos duplicados.

## O que foi praticado/aprendido
- Estruturas de dados em Python, especificamente sets.
- Conversão entre listas e conjuntos.
- Lógica básica de programação para processamento de coleções.

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Certifique-se de ter o Python instalado em seu sistema.
2. Salve o código acima em um arquivo chamado `atividade7.py`.
3. Abra o terminal ou prompt de comando.
4. Navegue até o diretório onde você salvou o arquivo.
5. Execute o projeto digitando o seguinte comando:
   ```sh
   python atividade7.py
   ```
6. O programa imprimirá os valores únicos da lista original.
