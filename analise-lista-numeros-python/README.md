# 📌 Análise e Busca de Números em Lista

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza uma análise de uma lista de números, processando valores individualmente e identificando o primeiro número ímpar divisível por 7 encontrado.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de manipulação de listas, estruturas de repetição e controle de fluxo, desenvolvendo uma solução capaz de percorrer dados e executar ações específicas conforme determinadas condições.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Percorre uma lista de números predefinida;
- Verifica se cada número é par ou ímpar;
- Ignora números pares utilizando o comando `continue`;
- Analisa números ímpares em busca de valores divisíveis por 7;
- Interrompe a execução ao encontrar o primeiro número correspondente;
- Exibe os números processados durante a análise.

Funcionamento:

- Números pares:
  - São ignorados pelo sistema.

- Números ímpares:
  - São analisados normalmente.

- Número ímpar divisível por 7:
  - É identificado e encerra a busca.

## ✨ Funcionalidades

- Percorrer elementos de uma lista;
- Filtrar valores conforme condições;
- Ignorar elementos específicos;
- Encontrar um valor utilizando busca sequencial;
- Controlar execução de loops.
