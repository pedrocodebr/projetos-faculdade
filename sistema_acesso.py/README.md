# 📌 Sistema de Controle de Acesso

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema simula um processo de validação de acesso de usuários, verificando informações como idade, matrícula e autorização especial para determinar se o acesso será permitido ou negado.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como entrada e saída de dados, manipulação de strings e estruturas condicionais, por meio da criação de um sistema capaz de tomar decisões automaticamente com base nas informações fornecidas pelo usuário.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita a idade do usuário;
- Verifica se o usuário está matriculado;
- Analisa se o usuário possui idade suficiente para acesso;
- Caso o usuário seja menor de idade, solicita uma autorização especial;
- Define automaticamente se o acesso será permitido ou negado;
- Exibe uma mensagem informando o resultado da validação.

## ✨ Funcionalidades

- Validação de matrícula do usuário;
- Verificação de idade mínima para acesso;
- Solicitação de autorização especial para menores de idade;
- Controle de permissão de acesso;
- Exibição de mensagens de aprovação ou bloqueio.

## 🛠️ Tecnologias Utilizadas

- Python 3

## 🧠 Conceitos Praticados

- Variáveis;
- Entrada e saída de dados;
- Conversão de tipos (`int`);
- Manipulação de strings:
  - `.strip()`
  - `.lower()`
- Estruturas condicionais (`if` e `else`);
- Operadores relacionais;
- Operadores de comparação;
- Lógica de decisão e validação de dados.
