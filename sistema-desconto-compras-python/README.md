# 📌 Sistema de Cálculo de Desconto em Compras

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema simula um processo de vendas, calculando automaticamente descontos em compras que ultrapassam um determinado valor e gerando o total final das vendas realizadas.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, operações matemáticas e regras de negócio, desenvolvendo uma solução capaz de processar compras e aplicar descontos automaticamente conforme critérios definidos.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita valores de compras ao usuário;
- Verifica se a compra possui direito a desconto;
- Aplica desconto de 10% em compras acima de R$ 100,00;
- Conta a quantidade de compras que receberam desconto;
- Soma todos os valores finais das compras;
- Exibe o total acumulado e a quantidade de descontos aplicados.

Regras aplicadas:

- Compras acima de **R$ 100,00**:
  - Recebem **10% de desconto**.

- Compras abaixo ou iguais a **R$ 100,00**:
  - Mantêm o valor original.

- Valor **-1**:
  - Encerra o cadastro das compras.

## ✨ Funcionalidades

- Cadastro contínuo de compras;
- Aplicação automática de descontos;
- Contagem de compras com desconto;
- Cálculo do valor total final;
- Encerramento através de condição de parada;
- Processamento de valores financeiros.
