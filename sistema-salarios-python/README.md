# at11-python

## Descrição
Este projeto calcula a média dos salários fornecidos pelo usuário até que um valor negativo seja inserido.

## Objetivo
O objetivo deste projeto é ensinar o uso de loops `while`, condicionais e cálculos simples em Python. Também visa familiarizar com a entrada de dados do usuário e o processamento básico de informações numéricas.

## O que foi praticado/aprendido
- Uso de loop `while` para repetição indeterminada
- Condicionais (`if`, `else`, `continue`) para controle de fluxo
- Cálculo de média
- Entrada e processamento de dados do usuário

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Clone o repositório em sua máquina:
   ```bash
   git clone https://github.com/seu-usuario/at11-python.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd at11-python
   ```
3. Execute o script `at11.py` usando Python:
   ```bash
   python at11.py
   ```

Aguarde a solicitação de entrada de salário e digite valores positivos para calcular a média. Para encerrar o programa, insira um valor negativo.
