# att 6

## Descrição
Este projeto contém uma série de exercícios práticos em Python, abrangendo diversas operações com listas, dicionários, tuplas e conjuntos, além de contar com o cálculo de médias, identificação de cursos mais caros, filtragem de participantes por idade e confirmação, cálculo de valor total de estoque, análise de letras em uma palavra, operações com conjuntos e classificação de notas.

## Objetivo
O objetivo deste projeto é proporcionar um ambiente prático para a aprendizagem e aplicação dos conceitos teóricos de programação em Python. O desenvolvedor poderá consolidar seus conhecimentos sobre estruturas de dados, controles de fluxo, funções e manipulação de strings.

## O que foi praticado/aprendido
Neste projeto foram praticados os seguintes aspectos:
- Estruturas de controle (`for`, `if`, `else`, `continue`, `break`)
- Operações em listas, dicionários e conjuntos
- Funções built-in do Python
- Cálculos simples de médias e totais
- Manipulação de strings e contagem de caracteres

## Tecnologias utilizadas
O projeto foi desenvolvido usando o seguinte software:
- Python 3.x
- Ambiente de execução padrão do Python (sem dependências adicionais)

## Como executar
1. Faça um fork deste repositório no seu GitHub.
2. Clone o repositório em sua máquina local utilizando `git clone https://github.com/seu-usuario/att-6.git`.
3. Acesse o diretório do projeto com `cd att-6`.
4. Execute cada arquivo Python individualmente usando `python nome_do_arquivo.py`.

Por exemplo, para executar o primeiro exercício:
```sh
python atividade1.py
```

Repetir o passo 4 para todos os arquivos disponíveis no repositório.
