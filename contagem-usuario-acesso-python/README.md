# atividade8-python

## Descrição
Este projeto contém um script Python que analisa uma lista de acessos e conta o número de vezes que cada usuário aparece na lista.

## Objetivo
O objetivo deste projeto era praticar a manipulação de listas, dicionários e laços de repetição em Python.

## O que foi praticado/aprendido
- Manipulação de listas
- Uso de dicionários para armazenar contagens
- Estrutura `for` para iterar sobre listas
- Condicional `if-else` para verificar a existência de chaves em um dicionário

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Salve o conteúdo do código em um arquivo chamado `atividade8.py`.
2. Abra uma terminal ou prompt de comando.
3. Navegue até o diretório onde o arquivo foi salvo.
4. Execute o script com o seguinte comando:
   ```bash
   python atividade8.py
   ```

O output esperado será um dicionário mostrando a contagem de acessos por usuário, conforme demonstrado no código fornecido.
