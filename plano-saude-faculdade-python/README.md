# PROJETO FACULDADE

## Descrição
Este projeto é uma aplicação simples que simula um sistema para escolher e calcular o custo de um plano de saúde. O usuário pode selecionar um dos planos disponíveis (Ouro, Diamante, Prata, Esmeralda), informar a idade, sexo e número de dependentes do cliente. O programa então calcula o valor total do plano com base em regras específicas (como faixa etária, sexo, e quantidade de dependentes) e exibe os detalhes do plano escolhido.

## Objetivo
O objetivo deste projeto foi criar uma aplicação que envolva a leitura de entradas do usuário, condições if-else, loops while, funções e manipulação de dados básicas em Python. Além disso, o projeto visa demonstrar como estruturar um código modular para facilitar a manutenção e expansão.

## O que foi praticado/aprendido
Durante o desenvolvimento deste projeto foram aplicados conceitos técnicos como:
- Estrutura condicional if-else
- Loops while
- Funções definidas pelo usuário
- Manipulação de entrada do usuário
- Operações matemáticas e arredondamento

Além disso, foi praticada a lógica de programação para resolver problemas simples com base em regras específicas.

## Tecnologias utilizadas
- Linguagem de programação: Python 3.x
- Bibliotecas utilizadas: Nenhuma biblioteca externa foi necessária para este projeto

## Como executar
Para rodar o projeto, siga os passos abaixo:

1. Certifique-se de que você tem Python instalado em sua máquina. Você pode baixá-lo e instalar do [site oficial do Python](https://www.python.org/downloads/).
2. Crie um novo arquivo chamado `FACUL.py` e copie o código acima neste arquivo.
3. Abra um terminal ou prompt de comando.
4. Navegue até o diretório onde você criou o arquivo `FACUL.py`.
5. Execute o script digitando o seguinte comando:

```bash
python FACUL.py
```

6. Siga as instruções na tela para escolher o plano, informar a idade, sexo e número de dependentes.

E pronto! Você terá um resultado exibido com o valor total do plano de saúde calculado conforme as regras definidas no código.
