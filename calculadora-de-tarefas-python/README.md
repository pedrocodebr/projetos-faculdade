# at15-python

## Descrição
Este projeto é uma aplicação simples em Python que calcula a média de um conjunto de números informados pelo usuário e conta quantos desses números estão acima da média.

## Objetivo
O objetivo deste projeto foi praticar o uso de estruturas de repetição, condicionais, listas e operações matemáticas em Python. Além disso, serviu como uma introdução à leitura e interpretação de código existente.

## O que foi praticado/aprendido
- Estrutura de repetição `while`
- Condicional `if`, `elif` e `else`
- Listas em Python
- Operações matemáticas básicas
- Funções built-in como `sum()` e `len()`

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Certifique-se de ter o Python instalado na sua máquina.
2. Baixe ou clone este repositório no seu computador.
3. Navegue até a pasta onde foi salvo o arquivo `at15.py`.
4. Execute o script usando o comando:
   ```
   python at15.py
   ```

A aplicação irá solicitar que você insira um número de tarefas, e em seguida receberá valores para cada tarefa. Após inserir todos os valores, o programa exibirá a média dos números e quantos desses estão acima da média.
