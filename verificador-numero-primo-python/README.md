# 📌 Verificador de Número Primo

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema verifica se um número informado pelo usuário é primo, utilizando um algoritmo de validação baseado em divisibilidade.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, operadores matemáticos e controle de fluxo, desenvolvendo uma solução capaz de analisar um número e determinar automaticamente se ele possui apenas dois divisores ou não.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita um número inteiro ao usuário;
- Verifica se o número informado é menor ou igual a 1;
- Testa possíveis divisores entre 2 e o número informado;
- Identifica se existe algum divisor além de 1 e dele mesmo;
- Exibe o resultado da análise.

Critério utilizado:

- Um número primo possui apenas dois divisores:
  - 1;
  - O próprio número.

Exemplos:

- **7** → É primo.
- **10** → Não é primo.

## ✨ Funcionalidades

- Verificação automática de números primos;
- Teste de divisibilidade;
- Tratamento de números menores ou iguais a 1;
- Interrupção do processo ao encontrar um divisor;
- Exibição do resultado da validação.
