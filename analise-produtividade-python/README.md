# 📌 Sistema de Análise de Produtividade

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite registrar a quantidade de tarefas realizadas e gera uma análise comparando os valores cadastrados com a média obtida.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de listas, estruturas de repetição e análise de dados, desenvolvendo uma solução capaz de armazenar informações, calcular indicadores e identificar valores acima da média.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita a quantidade de tarefas realizadas;
- Armazena os valores válidos em uma lista;
- Permite encerrar o cadastro informando um valor negativo;
- Ignora valores iguais a zero;
- Calcula a média das tarefas registradas;
- Identifica quantos registros ficaram acima da média;
- Exibe um resumo dos resultados.

Regras aplicadas:

- Valores maiores que **0**:
  - São armazenados para análise.

- Valor igual a **0**:
  - É ignorado pelo sistema.

- Valor negativo:
  - Finaliza a entrada de dados.

## ✨ Funcionalidades

- Cadastro de múltiplos registros;
- Armazenamento de dados utilizando listas;
- Cálculo automático da média;
- Comparação dos valores registrados;
- Contagem de valores acima da média;
- Geração de análise de produtividade.
