# media-estudantil

## Descrição
Este projeto calcula a média das três notas fornecidas pelo usuário e determina se o aluno foi aprovado, em recuperação ou reprovado com base na média obtida.

## Objetivo
O objetivo deste projeto era criar um programa simples em Python que pudesse calcular a média de três notas e informar o status do aluno com base nessa média.

## O que foi praticado/aprendido
No desenvolvimento deste projeto, foram praticados conceitos básicos de programação, como:
- Recebimento de entrada do usuário via `input()`
- Conversão de tipos de dados usando `float()`
- Cálculo e armazenamento de valores em variáveis
- Estruturas condicionais (`if`, `elif`, `else`) para tomar decisões com base nos valores calculados

## Tecnologias utilizadas
O único recurso técnico utilizado neste projeto foi a linguagem de programação Python.

## Como executar
Para executar este programa, siga os passos abaixo:

1. **Baixe o arquivo**: Faça o download do arquivo `media-estudantil.py` deste repositório.
2. **Abra um terminal**: Navegue até o diretório onde o arquivo foi salvo.
3. **Execute o script**: Digite o seguinte comando no terminal:
   ```
   python media-estudantil.py
   ```
4. **Seguindo as instruções**: O programa irá solicitar que você insira três notas. Após a inserção das notas, o programa informará se o aluno foi aprovado, em recuperação ou reprovado com base na média calculada.

Este projeto é perfeito para iniciantes em programação Python, oferecendo uma introdução prática aos fundamentos do lógico de programação e ao uso de condicionais.
