# 📌 Sistema de Cálculo de Média de Números

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza o cálculo da média de números informados pelo usuário, utilizando uma estrutura de repetição para receber múltiplos valores até que uma condição de encerramento seja informada.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, variáveis acumuladoras e controle de fluxo, desenvolvendo uma solução capaz de processar uma sequência de valores e calcular automaticamente sua média.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números ao usuário;
- Armazena a soma dos valores positivos informados;
- Conta a quantidade de números positivos digitados;
- Encerra a entrada de dados quando um número negativo é informado;
- Calcula a média dos números positivos;
- Exibe o resultado final.

Funcionamento:

- Números positivos:
  - São adicionados ao cálculo da média.

- Número negativo:
  - Finaliza a entrada de dados.

- Caso nenhum número positivo seja informado:
  - O sistema informa que não existem valores para calcular a média.

## ✨ Funcionalidades

- Leitura de múltiplos valores;
- Controle de repetição até uma condição de parada;
- Soma acumulativa dos números;
- Contagem de valores inseridos;
- Cálculo automático da média;
- Tratamento de ausência de dados.
