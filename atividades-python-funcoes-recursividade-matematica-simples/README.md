# att 5

## Descrição
Este projeto contém uma série de atividades práticas em programação Python, abrangendo conceitos básicos a mais avançados.

## Objetivo
A intenção deste projeto é proporcionar prática na resolução de problemas utilizando diferentes paradigmas de programação em Python, desde operações matemáticas até manipulação de listas e funções recursivas.

## O que foi praticado/aprendido
- Funções personalizadas com parâmetros e retorno
- Estruturas de controle como `if`, `elif` e `else`
- Listas e suas operações
- Compreensão de listas
- Expressões lambda
- Filtragem de dados usando `filter`
- Mapamentos de dados usando `map`
- Funções recursivas
- Operadores de comparação e lógicos
- Exceções personalizadas

## Tecnologias utilizadas
- Python 3.x

## Como executar
1. Clone o repositório no seu computador.
2. Abra um terminal ou prompt de comando.
3. Navegue até a pasta do projeto.
4. Execute os arquivos individuais, por exemplo:
   ```sh
   python atividade1.py
   ```

Cada arquivo contém uma atividade específica e pode ser executado separadamente para ver o resultado.
