# atividade1-python

## Descrição
Este projeto calcula a média de uma lista de notas e contabiliza quantas delas são aprovatórias (maiores ou iguais a 7).

## Objetivo
O objetivo deste projeto é praticar a manipulação de listas, cálculos simples e estruturas de controle em Python.

## O que foi praticado/aprendido
- Operações básicas de cálculo (`soma`, `media`)
- Estrutura de loop `for`
- Condicional `if` para verificar condições
- Listas no Python
- Operação `len()` para obter o tamanho de uma lista

## Tecnologias utilizadas
- Linguagem: Python

## Como executar
1. Certifique-se de ter Python instalado na sua máquina.
2. Salve o código acima em um arquivo chamado `atividade1.py`.
3. Abra o terminal ou prompt de comando e navegue até a pasta onde o arquivo foi salvo.
4. Execute o script com o seguinte comando:
   ```bash
   python atividade1.py
   ```
5. O programa exibirá a média das notas e quantas delas são aprovatórias.
