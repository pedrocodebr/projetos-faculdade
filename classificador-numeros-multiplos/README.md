# 📌 Classificador de Números Divisíveis por 3 e 5

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema realiza a identificação de números divisíveis simultaneamente por 3 e 5 dentro de um intervalo definido, utilizando estruturas de repetição e operadores matemáticos.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos fundamentais da linguagem Python, como estruturas de repetição, operadores aritméticos e estruturas condicionais, desenvolvendo uma solução capaz de percorrer uma sequência de números e identificar valores que atendem a determinados critérios.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Percorre números de 1 até 100;
- Verifica se cada número é divisível por 3;
- Verifica se o mesmo número também é divisível por 5;
- Exibe somente os números que atendem aos dois critérios.

Resultado esperado:

- Números múltiplos de 3 e 5 encontrados no intervalo:
  - 15
  - 30
  - 45
  - 60
  - 75
  - 90

## ✨ Funcionalidades

- Percorrer uma sequência numérica automaticamente;
- Identificar múltiplos de determinados valores;
- Aplicar condições de filtragem;
- Exibir apenas os resultados válidos.
