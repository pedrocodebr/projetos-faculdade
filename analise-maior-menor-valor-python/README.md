# 📌 Sistema de Análise de Maior e Menor Valor

## 📖 Descrição

Este projeto foi desenvolvido durante os estudos da disciplina de Programação em Python do curso de **Análise e Desenvolvimento de Sistemas (ADS)**. O sistema permite que o usuário informe diversos valores numéricos e realiza automaticamente a identificação do maior e do menor número digitado.

## 🎯 Objetivo

O objetivo deste projeto foi aplicar conceitos de estruturas de repetição, variáveis de controle e comparação de valores, desenvolvendo uma solução capaz de analisar uma sequência de dados com quantidade indefinida de entradas.

## 📝 Sobre o Projeto

O programa realiza as seguintes etapas:

- Solicita números ao usuário continuamente;
- Permite encerrar a entrada digitando a palavra **"fim"**;
- Armazena o maior valor encontrado;
- Armazena o menor valor encontrado;
- Exibe os resultados após o encerramento.

Funcionamento:

- Enquanto o usuário informar números:
  - O sistema compara cada valor com o maior e menor registrados;
  - Atualiza os valores quando necessário.

- Caso nenhum número seja informado:
  - O sistema informa que não existem valores para analisar.

## ✨ Funcionalidades

- Entrada de quantidade ilimitada de números;
- Encerramento personalizado através de comando;
- Identificação automática do maior valor;
- Identificação automática do menor valor;
- Tratamento de ausência de dados.
