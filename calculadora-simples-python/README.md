# atividade6-python

## Descrição
Este projeto realiza a soma de uma lista de valores inteiros.

## Objetivo
A intenção deste projeto era criar um programa simples em Python que calculasse e exibisse a soma de uma sequência de números.

## O que foi praticado/aprendido
- Conhecimentos básicos de estrutura de repetição (`for` loop)
- Operações aritméticas em Python
- Armazenamento e manipulação de dados em listas

## Tecnologias utilizadas
- Linguagem de programação: Python

## Como executar
1. Certifique-se de ter o Python instalado em sua máquina.
2. Crie um arquivo chamado `atividade6.py` e copie o código fornecido acima.
3. Abra um terminal ou prompt de comando.
4. Navegue até o diretório onde você salvou o arquivo.
5. Execute o seguinte comando:
   ```bash
   python atividade6.py
   ```
6. O programa exibirá a soma total dos valores na lista, que neste caso é `1000`.
